Recommendations

Actionable recommendations based on the website audit findings for embryo-ai.co.za

Rendering & Content

Server-Side Rendering

  • Adopt SSR or prerender on critical routes (Home, Services, Contact)
  • If staying SPA-only, integrate a prerender service at the CDN/edge
  • Ensure the initial HTML includes essential meta tags

Essential Meta Tags

  • <title>, <meta name="description">, <link rel="canonical">
  • Open Graph (og:title, og:description, og:image, og:url)
  • Twitter cards for social media sharing
  • JSON‑LD for Organization (and Services/LocalBusiness if applicable)

Crawlability

Essential Files

  • Add /robots.txt that allows core routes and references the sitemap
  • Publish /sitemap.xml covering all key pages
  • Submit sitemap in Google Search Console (GSC)
  • Verify internal linking structure and navigation

Performance (Mobile‑first in SA)

Image Optimization

  • Optimize hero/LCP media: modern formats (WebP/AVIF)
  • Correct intrinsic dimensions and aspect ratios
  • Preload or fetchpriority="high" for the LCP image

CSS and JavaScript

  • Ship critical CSS inline and defer non‑critical styles
  • Split JS bundles and defer non‑essential scripts
  • Cache headers via CDN; set long‑TTL with cache‑busting for immutable assets
  • Avoid blocking fonts; use font-display: swap and limit variants

Accessibility

  • Validate with axe and keyboard‑only testing
  • Ensure logical heading hierarchy (H1-H6)
  • Provide proper ARIA labels for controls and landmarks
  • Ensure visible focus states and sufficient color contrast
  • Provide text alternatives for images
  • Ensure menu/components are operable without a mouse

Security

HTTP Security Headers

  • Add / verify headers: HSTS, CSP (start with report‑only then enforce)
  • X‑Content‑Type‑Options, X‑Frame‑Options
  • Referrer‑Policy, and Permissions‑Policy

Platform Security

  • Verify forms go to HTTPS endpoints
  • Minimize and integrity‑check third‑party scripts
  • Apply strict content policies for Lovable platform
  • Version-pin all external dependencies

Quick Triage Checklist

  • [ ] SSR/prerender for Home, Services, Contact
  • [ ] robots.txt present and references sitemap
  • [ ] sitemap.xml valid and submitted to GSC
  • [ ] Title + Description + Canonical + OG/Twitter in initial HTML
  • [ ] LCP image optimized and prioritized
  • [ ] Basic accessibility checks pass (axe / keyboard)
  • [ ] Security headers deployed at the edge/origin
  • [ ] Forms & links smoke‑tested (tel:, mailto:, WhatsApp)
